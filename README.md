# -Library_Management-_System-OOPS-in-C++
## Project Description
A menu-driven Library Management System developed in C++ using Object Oriented Programming concepts.

## OOP Concepts Used
- Classes & Objects
- Constructors
- Encapsulation
- Inheritance
- Polymorphism
- File Handling

## Features
### Admin
- Add Book
- Remove Book
- Update Book
- View All Books

### User
- Search Book
- Issue Book
- Return Book

## Technologies Used
- C++
- STL Vector
- File Handling

## How to Run
1. Open Dev C++
2. Create main.cpp
3. Enter the code
4. Compile and Run
